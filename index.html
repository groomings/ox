commands :P / @btr for help

help
Shows all commands or by category. Also sends a .txt file.
Usage: .help
dm
Sends a message to someone's DMs.
Usage: .dm @user Hello!
setdmdumpchannel
Sets where DM dumps go.
Usage: .setdmdumpchannel 1234567890123
dumpdm
Dumps messages from a DM to a file.
Usage: .dumpdm @user 50
setphotodumpchannel
Sets where dumped photos should go.
Usage: .setphotodumpchannel 1234567890123
dumpphotos
Saves photos from a DM into a .zip file.
Usage: .dumpphotos @user 20
avatar
Gets your avatar or someone else's.
Usage: .avatar @user
serverinfo
Shows details about the current server.
Usage: .serverinfo
define
Gets a definition of a word.
Usage: .define meme
silenttyping
Toggles your typing status visibility.
Usage: .st on or .st off
calc
Opens calculator (Windows only).
Usage: .calc
zipcode
Gets info based on ZIP code.
Usage: .zipcode 90210
dnslookup
Checks domain DNS records.
Usage: .dnslookup google.com
react
Auto-reacts to a user's messages.
Usage: .react @user ðŸ˜‚
autoreact
Toggles auto-react mode.
Usage: .autoreact on / .autoreact off
disablereact
Turns off auto-react for a user.
Usage: .disablereact @user
stream
Changes your rich presence status.
Usage: .stream New Music
stopstream
Stops your streaming status.
Usage: .stopstream
GPT
Uses OpenAI to generate a response.
Usage: .GPT what's a haiku?
google
Returns a top Google search result.
Usage: .google discord bot tutorial
screenshot
Takes a screenshot of a site.
Usage: .screenshot https://discord.com
tiktok
Finds a TikTok video using a keyword.
Usage: .tiktok funny cat
prefix
Shows or changes the bot prefix.
Usage: .prefix or .setprefix !
uptime
Displays how long the bot has been online.
Usage: .uptime
restart
Restarts the bot (owner only).
Usage: .restart
vc
Joins a voice channel by ID.
Usage: .vc 1234567890
vcdisconnect
Disconnects from a voice channel.
Usage: .vcdisconnect
vcstatus
Shows your VC mute/deafen state.
Usage: .vcstatus
vcmute
Toggles your mute status in VC.
Usage: .vcmute
vcdeafen
Toggles your deafen status in VC.
Usage: .vcdeafen
password
Generates a random secure password.
Usage: .password
encrypt
Encrypts a message using Caesar cipher.
Usage: .encrypt hello 4
decrypt
Decrypts a Caesar-encrypted message.
Usage: .decrypt lipps 4
validateemail
Checks if an email is real or temporary.
Usage: .validateemail test@mail.com
validatenumber
Checks if a phone number is valid or fake.
Usage: .validatenumber +123456789
ipinfo
Looks up IP address details.
Usage: .ipinfo 8.8.8.8
reload
Reloads a module (dev only).
Usage: .reload commands.help
txt
Creates and sends a .txt file from your message.
Usage: .txt notes Hello there!
lock
Locks the current text channel.
Usage: .lock
unlock
Unlocks the channel so people can type again.
Usage: .unlock
copyfiles
Copies files from one folder to another (local).
Usage: .copyfiles ./source ./dest
extractzip
Extracts all files from an uploaded zip archive.
Usage: .extractzip
zip
Compresses your uploaded files into a .zip file.
Usage: .zip mybackup
