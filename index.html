Bot Help Commands

General

help: Displays this help message or category-specific commands
ping: Calculates and displays the bot's latency
about: Displays information about the self-bot
time: Displays the current time

DM

setdmdumpchannel: Sets the target channel ID for the dumpdm command
dumpdm: Dumps messages from a DM to a .txt file
dm: Sends a direct message to a user
