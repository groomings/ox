Bot Commands

help
Shows all commands or a specific category list.
Usage: .help or .help fun
dm
Sends a private message to a user.
Usage: .dm @user Hello!
mimic
Repeats a user's message or copies everything they say.
Usage: .mimic @user
purge
Deletes a bunch of messages fast.
Usage: .clear 100
8ball
Ask a yes/no question. It answers like magic.
Usage: .8ball Is pizza the best?
about
Shows info about this bot.
Usage: .about
define
Looks up the meaning of a word.
Usage: .define chill
calc
Opens calculator on Windows.
Usage: .calc
caption
Adds a caption to an attached image.
Usage: .caption When you realize it's Monday again
